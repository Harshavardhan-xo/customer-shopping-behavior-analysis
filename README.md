#  Customer Shopping Behavior Analysis | Python, MySQL & Power BI

##  Project Overview
This project analyzes customer shopping behavior to uncover purchasing patterns, sales distribution, and category performance using **Python (Pandas, NumPy)**, **MySQL**, and **Power BI**.

The goal of this project is to transform raw shopping transaction data into meaningful business insights through data cleaning, SQL-based analysis, and interactive dashboard visualization.

---

##  Objectives
- Analyze customer purchasing patterns
- Identify top-performing product categories
- Examine regional and time-based sales trends
- Build an interactive dashboard for business insights

---

##  Tools & Technologies Used

###  Python
- Pandas – Data cleaning and manipulation

###  MySQL
- Data aggregation
- Business queries using GROUP BY, ORDER BY, date functions

###  Power BI
- KPI Cards (Total Sales, Order Count, Average Sales)
- Category & Region Performance Charts
- Monthly/Yearly Trend Analysis
- Interactive slicers and filters

---

##  Dataset
- Dataset: Customer Shopping Behavior Data
- Format: CSV
- Contains transaction-level sales information including:
  - Customer details
  - Product category
  - Region
  - Order date
  - Sales amount

---

##  Project Workflow

### 1️ Data Cleaning (Python)
- Removed duplicates and handled missing values
- Converted date columns to proper datetime format
- Extracted month and year for trend analysis

### 2️ Data Analysis (MySQL)
- Calculated total and average sales
- Analyzed category-wise and region-wise performance
- Identified monthly and yearly sales trends
- Measured order distribution across segments

### 3️ Dashboard Development (Power BI)
- Designed interactive dashboard
- Implemented KPI tracking
- Added slicers for dynamic filtering
- Visualized sales trends and category performance

---

##  Key Insights
- Certain product categories contribute significantly to overall sales
- Regional performance varies across customer segments
- Sales trends show noticeable monthly patterns
- A small number of categories drive a large portion of revenue
